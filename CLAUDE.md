# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

This is a personal Git learning notes repository (Git学習メモ). Content is written in Japanese and covers POSIX shell commands, cross-platform compatibility, and Git fundamentals. The repository itself is used as a hands-on demonstration of Git operations.

## Structure

- `git-memo.md` — the single source of truth for all learning material
- `sample.txt` — a demo file (ignored by .gitignore and not tracked in practice)

## Working with This Repo

There is no build system, test suite, or package manager. All work involves editing Markdown.

When adding content to `git-memo.md`, follow the existing style: Japanese prose with structured sections, practical command examples, and platform compatibility notes (Windows/macOS/Linux/Git Bash).

The `.gitignore` excludes `sample.txt`, `Thumbs.db`, and `.DS_Store`.
